# Analisis-de-viajes-en-taxi-en-Chicago
●	En este proyecto se analizaron los datos de una empresa de viajes en Chicago durante noviembre del 2017. Usando SQL se extraen los datos de la web, revisé y corregí los datos para su análisis. 
●	Mediante pruebas de hipótesis en Python obtuve los indicadores de principales lugares o recorridos realizados, analizando el impacto que tiene un día concurrido con lluvia en el servicio general de taxis. 
Habilidades tecnológicas utilizadas:  SQL, Pyhton, pruebas de hipótesis, matplotlib
